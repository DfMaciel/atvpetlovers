Atividade 1 - Lista PetLovers

Como executar:

1º - Instale as dependências necessárias utilizando o comando no diretório /src:

```
npm install
```

2º - Compile a aplicação em TypeScript:

```
npx tsc
```

3º - Acesse os arquivos compilados:

``` 
cd /src/out/main/
```

4º - Execute a aplicação:

```
node app.js
```
